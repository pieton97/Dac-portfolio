Recipe randomizer

I built this using the Jikan API. Data is fetched in parallel so load and renders are usually under than a second. You can search by keywords or any of the preset buttons below for anime results. Clicking on an anime will pop up a modal with all of the information about that clicked anime and recommendations. Modals can be easily closed by clicking on the outside. (My List) feature allows user to add any animes to their list and it will save within local storage for their next visit. This website is highly responsive and looks amazing on most devices.
